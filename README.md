# RBL Bank Website

A simple, attractive website for RBL Bank featuring bank information, offers, card types, and a contact form.

## Features

- Responsive design
- Bank information section
- Exclusive offers display
- Card types showcase
- Contact form with basic validation

## Usage

1. Open `index.html` in your web browser.
2. Navigate through the sections using the menu.
3. Fill out the contact form to submit inquiries.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Images

All images are sourced from placeholder services for demonstration purposes. In a production environment, replace with actual bank-approved images.

## Troubleshooting

- Ensure all files (`index.html`, `styles.css`, `script.js`) are in the same directory.
- If the form doesn't submit, check the browser console for JavaScript errors.
