# Axis Bank Credit Card Website

A professional, responsive website for Axis Bank credit card applications featuring real product details and interactive features.

## 📁 Project Structure

```
axis-bank/
├── index.html          # Main HTML file with all sections
├── styles.css          # Comprehensive styling and responsive design
├── script.js           # JavaScript for interactivity
└── README.md           # This file
```

## 🎯 Features

### 1. **Navigation Bar**
- Sticky navbar with smooth scrolling links
- Mobile responsive hamburger menu
- Quick "Apply Now" button

### 2. **Hero Section**
- Eye-catching banner with call-to-action
- Professional branding and messaging

### 3. **Credit Card Products**
Four premium Axis Bank credit cards with real details:
- **Signature Credit Card** - Premium tier with 5% cashback
- **MY Unlimited Credit Card** - Premium 5% cashback on all online purchases
- **ESSENTIAL Credit Card** - Lifetime free with basic benefits
- **Business Credit Card** - For commercial use with higher limits

Each card displays:
- Annual fee information
- Welcome bonus details
- Credit limit range
- Key features and benefits
- Quick apply button

### 4. **Benefits Section**
Highlights 6 major benefits:
- Exclusive Rewards
- Cashback Offers
- Travel Benefits
- Security & Safety
- Shopping Benefits
- 24/7 Support

### 5. **Application Form**
Comprehensive form with fields for:
- Personal information (Name, Email, Phone, DOB)
- Identity details (PAN, Address, City, Pincode)
- Financial information (Annual Income)
- Card preference selection
- Form validation with helpful error messages

### 6. **FAQ Section**
Accordion-style frequently asked questions covering:
- Required documents
- Approval timeline
- Income requirements
- Fees and charges
- Reward points tracking
- Payment grace period

### 7. **Contact Section**
Multiple contact options:
- Phone number (24/7 support)
- Email address
- Physical office locations

## 🚀 How to Use

1. **Open the website**: Simply open `index.html` in any modern web browser
2. **Navigate**: Use the navigation bar to jump to different sections
3. **Explore Products**: Scroll through the credit cards and click individual cards for details
4. **View Benefits**: Check out what each card offers
5. **Apply**: Fill out the application form with your details
6. **FAQ**: Get answers to common questions
7. **Contact**: Find support options

## ✨ Interactive Features

- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Form Validation**: Real-time validation for:
  - Email format
  - Phone number (10 digits)
  - PAN format
  - Age verification (18+ years)
  - Pincode format
- **FAQ Accordion**: Click to expand/collapse answers
- **Success Modal**: Confirmation message after form submission
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Hover Effects**: Interactive card animations and button effects
